# PySketch - visual programming for Python Turtle

PySketch is a modern, web-based visual programming environment that bridges the gap between artistic creativity and coding. Draw directly on the canvas, and watch as your strokes are instantly converted into clean, executable Python Turtle code.

## ‚ú® Features

- **Visual Drawing Canvas**: Intuitive drawing interface with pen and eraser tools.
- **Instant Code Generation**: Real-time conversion of your drawing into `turtle` commands.
- **Layer Management**: multiple layers to organize your artwork (Background, Details, etc.).
- **Smart Simplification**: Algorithms that smooth your strokes for cleaner code output.
- **Customization**:
  - Adjustable stroke widths and colors
  - Customizable canvas background colors
  - Variable drawing speeds
- **Modern UI**: A sleek, dark-mode interface built for focus and ease of use.
- **Export**: Download your creation as a `.py` file to run locally.

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Solez-ai/PySketch.git
   cd PySketch/pysketch
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to start creating!
